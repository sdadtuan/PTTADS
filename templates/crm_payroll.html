Vân tay / khuôn mặt / Face ID

Trình duyệt không đọc trực tiếp cảm biến vân tay hay khuôn mặt. Dùng máy chấm công (ZKTeco MB20-VL, MB560-VL, Hikvision…) gửi dữ liệu về PTT qua mạng LAN.

ZKTeco MB20-VL (khuyến nghị — giao thức iClock PUSH)

  1. Cắm máy vào cùng mạng LAN với máy chủ PTT (Wi‑Fi hoặc dây).
  2. Trên máy: Menu → Comm. → Cloud Server / ADMS (tên menu tuỳ firmware).
  3. Đặt Server Address = IP máy chạy PTT, cổng 5050, đường dẫn /iclock/cdata.
  4. Bật Real-time push (nếu có) → Lưu → khởi động lại máy.
  5. Ghi Serial Number (SN) trên máy; tuỳ chọn đặt env CRM_ZKTECO_ALLOWED_SN=SN-máy-bạn để chỉ nhận từ máy đó.
  6. CRM → Nhân sự: với mỗi NV, điền Mã trên máy chấm công (PIN) trùng User ID trên MB20-VL (số đăng ký khuôn mặt).
  7. Đăng ký khuôn mặt trên máy → chấm thử → kiểm tra bảng Chấm công bên dưới.

URL máy MB20-VL: {{ zkteco_iclock_url }}

Máy gọi GET /iclock/getrequest và POST /iclock/cdata?SN=…&table=ATTLOG — PTT tự parse giờ vào/ra. Không cần cài ZKBio Time nếu máy push trực tiếp được.

API JSON (script / gateway khác)

  1. Đặt env CRM_ATTENDANCE_DEVICE_KEY = chuỗi bí mật.
  2. POST JSON kèm header X-CRM-Device-Key hoặc Authorization: Bearer ….

Endpoint: {{ attendance_device_url }}

{% if device_key_configured %}Đã có khóa CRM_ATTENDANCE_DEVICE_KEY trên server.{% else %}Chưa có CRM_ATTENDANCE_DEVICE_KEY — API thiết bị trả 503 cho đến khi cấu hình.{% endif %}

Ví dụ JSON: {"pin":"0001","punched_at":"2026-05-05T08:30:00","kind":"auto","verify":"fingerprint"} — hoặc work_date + time. kind: in / out / auto.
